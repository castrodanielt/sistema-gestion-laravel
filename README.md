# Sistema Gestión Empresarial
Aplicación empresarial desarrollada con Laravel 13.

## Descripcion
Proyecto orientado a la gestión de clientes, productos y procesos comerciales

## Tecnologías
- PHP
- Laravel 13
- MySQL
- Composer
- XAMPP

## Funcionalidades
- Entidad Cliente
- CustomerController
    - Listado de Customers
    - Creación de Customers
    - Editado de Customers
    - Soft Delete de Customers
